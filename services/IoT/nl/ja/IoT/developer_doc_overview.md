# {{site.data.keyword.iot_short}} の開発者資料
{: #developer_doc_overview}

コンテンツおよびサンプルは、アプリケーションを作成し、デバイスを {{site.data.keyword.iot_full}} 組織内で統合するために用意されています。

{:shortdesc}


## {{site.data.keyword.iot_short}} の開発
{: #goals}
デバイスをセットアップして {{site.data.keyword.iot_short}} に接続し、関心があるリアルタイム・データを取得して分析する方法はいくつかあります。以下のタスクを実行するための情報が提供されます。

-  アプリケーションの作成と管理
-  デバイスの作成、接続、管理
-  デバイス管理の拡張
-  ゲートウェイの作成と管理
-  デバイス・データの検索


## アプリケーション開発
{: #app-dev}

サポートされる以下のプロトコルと規格を使用して、{{site.data.keyword.iot_short}} でアプリケーションを作成し管理できます。

- MQTT メッセージング・プロトコル
- Python
- Node.js
- Java
- C#

## デバイスの開発
{: #device-dev}

サポートされる以下のプロトコルと規格を使用して、{{site.data.keyword.iot_short}} でデバイスを作成し管理できます。

- MQTT メッセージング・プロトコル
- Python
- Node.js
- Java
- C#
- Embedded C
- mBed C++

## ゲートウェイの開発
{: #gateway-dev}
デバイスがインターネットに直接接続できない場合、提供される情報を使用して、{{site.data.keyword.iot_short}} 組織内でデータを検索しアプリケーション送信するためのゲートウェイ・デバイスを作成します。
{{site.data.keyword.iot_short}} のゲートウェイ開発では、MQTT メッセージング・プロトコルがサポートされています。
