# 管理用户角色
上次更新时间：2016 年 9 月 16 日
{: .last-updated}

用户角色指示该用户有权执行的操作。您可以通过使用 {{site.data.keyword.iot_full}} 仪表板或使用 {{site.data.keyword.iot_short_notm}} API 来更改用户的分配角色。

## 通过使用仪表板更改用户角色

1. 从 {{site.data.keyword.iot_short_notm}} 仪表板，在左侧导航栏中选择**访问** > **成员**。
2. 单击要更新的用户旁的**编辑**图标。
3. 选择角色，然后单击**完成**。
